## 🎯 **How to Use Recovery Features**

### For Academic Officers (GUI):
1. **Login as Officer**: `officer` / `officer123`
2. **Go to Recovery Plans Tab**
3. **Click "Identify At-Risk Students"**
4. **View generated recovery plans and statistics**

### Recovery Plan Creation:
```java
// Automatic creation for ineligible students
RecoveryManager recoveryManager = new RecoveryManager();
List<RecoveryPlan> plans = recoveryManager.generateRecoveryPlansForIneligibleStudents(students, officerId);

// Manual creation
RecoveryPlan plan = recoveryManager.createRecoveryPlan(studentId, courseCode, courseName, createdBy);
```

### Eligibility Checking:
```java
EligibilityChecker checker = new EligibilityChecker();
String report = checker.getEligibilityReport(student);
boolean eligible = checker.isEligibleForProgression(student);
```
